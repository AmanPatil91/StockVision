# AI Stock Intelligence Platform

An enterprise-grade AI-powered stock analysis and prediction platform focused on swing trading and breakout investing.

---

## Vision

Create an intelligent investment platform that combines:

- Technical Analysis
- Fundamental Analysis
- News Intelligence
- Machine Learning
- Explainable AI

to predict medium-term (1–6 month) stock movements.

---

# Changelog

## Version 0.2.0

- Phase 2: Technical Analysis Engine (Completed)
  - Backend: Added Technical Indicator API
  - Backend: Integrated `ta` library for robust indicator math
  - Backend: Supported indicators: EMA, SMA, VWAP, RSI, MACD, ADX, ATR, Bollinger Bands, OBV, MFI, CMF, Pivot Points
  - Backend: Added Technical Summary API with ratings for Trend, Momentum, Volatility, and Volume
  - Backend: Automated generation of professional technical signals (Golden Cross, Death Cross, RSI Extremes, Breakouts, MACD Crosses)
  - Frontend: Built Indicator Store using Zustand for fast state management
  - Frontend: Built localized robust calculations engine to draw historical lines
  - Frontend: Built Responsive Indicator Panel UI
  - Frontend: Added interactive Dynamic Chart Overlays (SMA, EMA, VWAP, BB)
  - Frontend: Built Synchronized Oscillator charts for RSI and MACD
  - Frontend: Integrated Technical Summary UI Card into Dashboard
  - Global: Reusable AI-friendly JSON schema output. Performance optimization (React Memo, SWR/Query Caching).
